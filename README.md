# UseAlibi — Releases

Public distribution channel for **UseAlibi**. This repository holds the built
Windows installers (as GitHub Release assets) and the `version.json` manifest
the app reads to check for updates.

- **Download the latest:** see [Releases](../../releases/latest).
- `version.json` — the update manifest (latest version + download URL + notes).

The UseAlibi source code is proprietary and lives in a separate private repository.
